## Веб-приложение для библиотеки

### Общая информация

Данное CRUD-приложение позволяет вести цифровой учёт книг в библиотеке.

Основные функции:

Управление списком чителей — добавление / просмотр / редактирование / удаление.

Управление списком книг — добавление / просмотр / редактирование / удаление / выдача книги / возврат книги.

Можно отслеживать у какого читателя сейчас находится определённая книга, а также получать список всех книг, взятых читателем.

Отношение между сущностями Person и Book — One to Many

Техническое задание:
https://swiftbook.org/system/uploads/lecture_material/material/866/Project1_TZ.pdf

### Технологии

Движок веб-приложения: Spring Web MVC

База данных: PostgreSQL

Взаимодействие с БД: JdbcTemplate

Фронт: HTML / CSS

Шаблонизатор: Thymeleaf

Система сборки: Maven

Структура проекта: Maven-Archetype-WebApp
